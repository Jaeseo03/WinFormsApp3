### 🚀 주요 구현 기능
본 프로젝트는 C# WinForms를 활용하여 제작되었으며, 다음의 핵심 기능들을 포함하고 있습니다.

1. **배경색 랜덤 변경**: [배경색상 변경] 버튼 클릭 시 무작위 RGB 값으로 배경색이 변화합니다.
2. **GitHub 연동**: [GitHub] 버튼 클릭 시 개발자의 리포지토리로 자동 연결됩니다.
3. **사진 전환 기능 (Toggle)**: [사진 변경] 버튼 클릭 시 명함 사진이 두 종류로 번갈아 가며 바뀝니다.
4. **마우스 인터랙션**: 사진 위에 마우스를 올리면 `BorderStyle`이 변경되어 시각적 피드백을 제공합니다.

### 💡 구현 시 어려웠던 점 및 해결 방법
- **프로젝트 명칭 및 네임스페이스 변경**: 초기 프로젝트명이 규칙과 달라 수정하는 과정에서 오류가 발생했으나, Visual Studio의 '이름 바꾸기' 기능과 네임스페이스 일괄 수정을 통해 해결하였습니다.
- **Git 계정 설정**: 로컬 Git 설정과 GitHub 계정이 달라 커밋 작성자가 다르게 표시되는 문제가 있었으나, 전역 설정을 다시 확인하고 프로젝트 구조를 정리하며 Git 사용법을 익혔습니다.
